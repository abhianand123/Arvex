# Arvex

<img src="./assets/outertune.webp" height="88" alt="Arvex app icon">

A Modern, Material 3 YouTube Music client & local music player for Android. Stylish, Powerful, Yours.

> [!WARNING]
> Arvex is currently in active development.

## Features

Arvex is a supercharged fork of [InnerTune](https://github.com/z-huang/InnerTune). This app is both a local media player, and a YouTube Music client.

- **Modern Aesthetic**: Sleek Material 3 Design that feels premium.
- **YouTube Music Client**:
    - Download songs for offline playback
    - Ad-free background playback
    - Account synchronization
- **Local Music Player**:
    - Plays MP3, OGG, FLAC, and more
    - Custom tag extractor for accurate metadata
- **Advanced Audio**:
    - Audio normalization
    - Tempo/Pitch adjustment
    - Audio effects
- **Lyrics**: Synchronized lyrics (LRC, TTML) and Karaoke support
- **Android Auto**: Full support
- **Chromecast**: Stream music to Chromecast devices seamlessly
- **Compatibility**: Android 8 (Oreo) and higher
